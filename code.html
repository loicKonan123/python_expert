Peux-tu m'expliquer comment fonctionnent les list comprehensions en Python avec un exemple complexe ?

Certainement. Les **compréhensions de liste** offrent une syntaxe concise pour créer des listes. Elles consistent en des crochets contenant une expression suivie d'une clause `for`, puis de zéro ou plusieurs clauses `for` ou `if`.

Voici un exemple illustrant une compréhension de liste imbriquée pour aplatir une matrice tout en filtrant les nombres pairs :

matrix_filter.py
# Une matrice 3x3 matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]] # Aplatir et filtrer les pairs flattened_evens = [ num for row in matrix for num in row if num % 2 == 0 ] print(flattened_evens) # Output: [2, 4, 6, 8]
